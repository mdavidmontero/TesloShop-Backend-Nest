# Teslo Api

1. Clonar proyecto
2. Instalar dependencias

```
npm install
```

3. Clonar el archivo .env.template en .env
4. Editar el archivo .env
5. Levantar la base de datos

```bash
docker-compose up -d
```

6. Ejecutar SEED

```bash
  http://localhost:3000/api/seed
```

6. Levantar:

```bash
   npm run start:dev
```
